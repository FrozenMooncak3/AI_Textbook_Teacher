# 最优学习路径指南

**用途**：任何需要系统吸收教材型内容（教科书 / 技术文档 / 课件）时的通用路径
**最后更新**：2026-04-13
**依据来源**：`docs/research/` 下 10 份调研（ICAP / Testing Effect / Bastani PNAS 2025 / Gagne 9 Events 等）

---

## 核心路径（6 阶段）

```
Phase 0: 学前激活  →  Phase 1: 阅读原文  →  Phase 2: 教学/讲解
                                                    ↓
Phase 5: 间隔复习  ←  Phase 4: 模块测试  ←  Phase 3: 独立检索 (闭卷 Q&A)
```

**完整路径（推荐）**：6 阶段全走
**最小路径（时间紧）**：Phase 1 → Phase 3 → Phase 5 （可以砍 Phase 0 / 2 / 4，但**不能砍 3 和 5**）

---

## 每阶段做什么 + 为什么

### Phase 0：学前激活（2-5 分钟）

**做什么**：
- 看一眼本次要学的目标（3-5 个要点清单）
- 回答 1-2 道探测题（答错没关系，不记录成绩）

**为什么必要**：
- **Pretesting Effect** —— 2025 meta-analysis 显示效果 g=0.66
- 原理：先让大脑"撞一下"这个话题，后续阅读时激活的神经回路会更强
- 答错了反而效果更好——因为你知道了"我其实不会"，警觉度提升

**核心原则**：探测题是**认知准备**，不是考试。不评分、不记录、答错没关系。

---

### Phase 1：阅读原文（不限时）

**做什么**：
- 完整读一遍原文，不跳读
- 可以划重点、标不懂的地方，但不要停下来查资料

**为什么必要**：
- 这是所有后续阶段的基础。**AI 教学不是替代阅读，是在阅读基础上做深度加工**
- 阅读时的"浅加工"是为后续"深加工"铺垫，不能省

**核心原则**：**必须读完才能进入后续阶段**。没有跳过按钮。

---

### Phase 2：AI 教学 / 讲解（10-25 分钟/次）

**做什么**：
- 每次教 3-5 个相关知识点（一个"小主题"）
- AI 按知识点类型自动选择教学法：
  - **事实类**（定义、术语、数据）→ 直接讲解 + 示例
  - **概念类**（模型、框架、原理）→ 类比 + 详细阐述
  - **过程类**（步骤、方法、流程）→ 示范 + 引导练习
  - **分析类**（推理、推导、诊断）→ 认知学徒（AI 想给你看）
  - **评估类**（判断、权衡、比较）→ 案例 + 反事实讨论
- 教学深度分两档：
  - **轻量版（3 步）**：激活 → 讲解 → 快速确认（3-5 分钟/知识点）
  - **完整版（5 步）**：激活 → 讲解 → 举例 → 练习 → 迁移（8-12 分钟/知识点）

**为什么必要**：
- **ICAP Framework**（Chi & Wylie 2014）：Interactive（互动）> Constructive（建构）> Active（主动）> Passive（被动）
- AI 对话式教学属于 Interactive 层，是最高效的学习形式
- **VanLehn 2011 元分析**：AI 的 step-level feedback 效果量 d=1.0，超过人类 tutor 的 d=0.79

**核心原则**：
- 教学深度按知识重要度选：次要知识点轻量，核心概念完整
- 教学过程中有 AI 搭梯子（提问、类比、示例），这是"工作记忆"训练

**⚠️ 关键陷阱**：
- **Bastani PNAS 2025 研究**：AI 教学如果"过度帮助"（直接给答案），闭卷测试成绩会 **-17%**
- 只有**带护栏的 AI 教学**（Socratic 引导、不给答案）才有 +127% 的练习效果
- 所以教学时 AI 应该多反问、少直接给结论

---

### Phase 3：独立检索 Q&A（10-15 分钟/次）

**做什么**：
- 3-10 道自适应题目（按答题表现动态调整难度）
- 题型按知识点类型匹配（见下表）
- 一次一题，答完立即看反馈
- **必须闭卷**：不能查看原文、笔记、教学对话记录

| 知识点类型 | 推荐题型 |
|---|---|
| 事实类 | 单选 / 简答 |
| 概念类 | Teach-back（用自己话解释）+ 预测题 |
| 过程类 | 简答 + 错误检测（给一段错的步骤让你找问题） |
| 分析类 | 预测 + 比较（A 和 B 哪个更合理） |
| 评估类 | 比较 + 预测（权衡多个选项） |

**为什么必要**：
- **Testing Effect** —— 2 份 meta-analysis 一致：
  - Adesope 2017: g=0.51
  - Rowland 2014: g=0.50
- 核心原理：**学懂 ≠ 学会**
  - 教学时"听懂了"是**工作记忆**（临时缓存）
  - QA 强制你从**长期记忆**里检索调取 —— 这是一次质变
  - 没做过 QA 的知识，第二天就忘 60-80%

- **Pan & Rickard 2018**：跨题型检索（教学一种题型，考试另一种）效果量 d=0.58，超过同题型 d=0.40 —— 所以 QA 题型要**和教学过程的题型不同**

**核心原则**：
- **闭卷是底线，不是推荐**。开卷 QA 效果减半
- **已答题不可修改**，只能往前 —— 防止"先交答案再翻书"作弊
- 错题自动归档到错题本（Phase 5 复习重点）

---

### Phase 4：模块测试（20-30 分钟/模块）

**做什么**：
- 整模块综合考试（混合题型：单选 / 计算 / 论述 / 评估）
- 80% 通过率（硬线，不是建议）
- **禁止查看笔记、教学记录、QA 记录**（界面上这些入口都要隐藏）

**为什么必要**：
- **Formative-Summative 三层评估理论**：
  - Layer 1（教学中嵌入式评估）：在 Phase 2 里
  - Layer 2（独立检索评估）：在 Phase 3 里
  - Layer 3（终结性评估）：就是 Phase 4 —— **不能合并到前两层**
- 终结性评估的作用是**模拟真实世界应用场景**（考试、实战、工作）—— 给你"这是最终检验"的心理压力

**核心原则**：
- 测试题型和 QA 不同（再次利用跨格式检索效应）
- 未过 80% 不能进入下一模块，必须回头复习错题再考

---

### Phase 5：间隔复习（持续，按提醒复习）

**做什么**：
- 按"小主题"（cluster，3-5 个相关知识点分组）抽题复习
- 间隔按算法动态调整：
  - 首次掌握 → 1 天后
  - 答对 → 3 天后 / 7 天后 / 21 天后 / 60 天后（逐渐拉长）
  - 答错 → 退回上一级间隔

**为什么必要**：
- **Ebbinghaus 遗忘曲线**：不复习的知识 1 天后保留 40%，30 天后保留 10%
- **Spaced Repetition**（SuperMemo / Anki 算法）：按遗忘曲线精准复习可把 30 天保留率拉到 90%+

**核心原则**：
- 复习按"小主题"，不按"单个知识点" —— 相关知识点一起复习效果更好（联想检索）
- 复习时同样**闭卷**

## 一次学习 session 的完整流程（学习者视角）

**场景**：要学一章新内容（约 10 个知识点，分成 2-3 个小主题）

1. **打开模块**（30 秒）
   - 看一眼目标清单："本章你将掌握 X / Y / Z"
   - 做 1-2 道探测题（Phase 0）

2. **阅读原文**（10-30 分钟，视内容长度）
   - 完整读一遍
   - 划重点、标不懂（Phase 1）

3. **AI 教学第一个小主题**（10-25 分钟）
   - 3-5 个相关知识点（Phase 2）
   - AI 按类型选教学法，按重要度选深度

4. **第一个小主题 QA**（10-15 分钟）
   - 闭卷，一次一题（Phase 3）

5. **重复 3-4，直到所有小主题完成**

6. **模块测试**（20-30 分钟）
   - 混合题型，闭卷，80% 过关（Phase 4）

7. **之后自动进入间隔复习**（1 天、3 天、7 天…）
   - 每次系统提醒你复习（Phase 5）

**单次学习时长**：约 40-90 分钟，建议拆成 2-3 次小 session（每次不超过 30 分钟，符合微学习原则）

---

## 不能违反的 5 条硬规则

1. **Phase 1 必须读完才能进后续** —— 不能跳过阅读直接 QA
2. **Phase 3 / 4 / 5 必须闭卷** —— 看原文/笔记的 QA 效果减半
3. **Phase 3 已答题不可修改** —— 防作弊
4. **Phase 4 通过线 80%** —— 硬线，不是软提示
5. **AI 教学不能直接给答案** —— Socratic 式引导，否则闭卷测试会 -17%

---

## 最小路径（时间极紧时）

如果真的没时间走完整 6 阶段，可以砍掉 Phase 0 / 2 / 4，但**至少要有**：

```
Phase 1 (阅读) → Phase 3 (闭卷 QA) → Phase 5 (间隔复习)
```

原因：
- 砍掉 Phase 0：损失 g=0.66 的激活效应，但可接受
- 砍掉 Phase 2：损失 Interactive 层，但 Phase 3 的闭卷 QA 仍能强化长期记忆
- 砍掉 Phase 4：损失终结性压力，但如果这是自学而非备考，影响小
- **绝对不能砍 Phase 3**：没有闭卷检索，学过等于没学
- **绝对不能砍 Phase 5**：没有间隔复习，一周后忘光
