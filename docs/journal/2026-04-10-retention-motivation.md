# 用户留存与学习动机

## 用户提出

需要思考如何留存用户、加强学习动机。当前产品只有功能流程，缺乏让用户"想回来继续学"的机制。

## 待探索方向

- 学习连续性（streak / 打卡）
- 进度可视化（让用户看到自己的成长）
- 复习提醒（推送 / 邮件）
- 学习数据回顾（本周学了多少、掌握了多少）
- 社交/竞争元素（排行榜、学习小组）— 注意 CLAUDE.md 禁止社交功能
- 成就系统 / 里程碑庆祝

## 状态

MVP 前需要确定至少一个核心留存机制。待 brainstorm 详细方案。
