# 笔记 + QA 联动

**类型**: idea
**状态**: parked
**日期**: 2026-03-29

## 想法

1. **阅读阶段写笔记**：用户在原文阅读时可以写笔记
2. **AI 生成结构化笔记**：模块学习完成后，AI 利用工具（类似 Obsidian）生成结构化学习笔记，并把用户自己的笔记整合进去
3. **QA 旁边弹出笔记要点**：做 QA 练习时，侧边栏可以弹出相关笔记要点，边做边学

## 评估

- 与现有 `reading_notes` 和 `module_notes` 表吻合，数据基础已有
- 属于 M5（体验打磨）范畴，或可独立为新里程碑
- 需要设计：笔记生成格式、侧边栏 UI、笔记与 KP 的关联方式
