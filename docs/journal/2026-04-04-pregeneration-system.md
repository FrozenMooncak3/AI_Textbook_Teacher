# 预生成系统

**类型**: idea
**状态**: parked
**日期**: 2026-04-04

## 想法

用户完成上一步后，后台悄悄预生成下一步内容。用户点进去直接看到结果，零等待。

| 触发时机 | 后台预生成 |
|----------|-----------|
| KP 提取完成 | → 预生成每个模块的读前指引 |
| 用户完成阅读 | → 预生成 QA 题目 |
| QA 全部完成 | → 预生成学习笔记 + 测试题 |
| 复习到期 | → 预生成复习题 |

## 评估

- 核心价值：消灭等待，从根本上解决用户流失风险
- 代价：后端需要后台任务机制，AI 费用略增（但用户迟早用到，非浪费）
- 成熟软件标准做法（Duolingo 等）
- 可逆性：容易反悔
